Week 2 Writing

Build a profile using identity from Week 1 and possession/age from Week 2.

1. Sentence warm-up

Reorder: ai / une clé / j'

Answer

J'ai une clé.

Complete: Samira ___ une question.

Answer

a

Complete: Nous ___ beaucoup de mots.

Answer

avons

2. Guided writing

  • Bonjour. Je m'appelle Alex.
  • Je suis étudiant / étudiante.
  • J'ai dix ans.
  • J'ai un sac et un cahier.
  • Je n'ai pas de téléphone.
  • J'ai une question. Madame Dupont a la réponse.

3. Controlled writing

Write 6 short sentences about Alex, Samira, or yourself. Use être, avoir, one object, one family word, and one negative sentence.

4. Writing test

Task: Write a short personal profile in French. Aim for 7-8 short sentences.

You may use: je m'appelle, je suis, j'ai ... ans, j'ai un sac / une clé / une sœur, je n'ai pas de téléphone, nous avons, merci, au revoir.

5. Checklist before reveal / submit

  • Did I use je m'appelle?
  • Did I use je suis for identity?
  • Did I use j'ai ... ans for age?
  • Did I add one object or family sentence?
  • Did I try one sentence with je n'ai pas?
  • Are my sentences short and clear?

6. Model answer hidden by default

Show model answer

Bonjour. Je m'appelle Samira. Je suis étudiante. J'ai six ans. J'ai une sœur. J'ai un sac et une clé. Je n'ai pas de téléphone. J'ai une question. Merci. Au revoir.
