# Content Generation Workflow

## Step 1
Read:
- learner_state
- curriculum_master
- grammar_map
- review_rules
- vocabulary_bank
- latest week summary

## Step 2
Decide:
- main grammar target
- review grammar target
- new vocabulary set
- recent review set
- long-term review set
- exact previous-week vocabulary and grammar that will be embedded in new examples, readings, listenings, writing prompts, and quiz questions
- reading goal
- listening goal
- writing goal
- testing mix

## Step 3
Enforce controlled language:
- list allowed known language
- list new language
- keep passages mostly within known language
- limit surprise vocabulary

## Step 4
Build in this order:
1. vocabulary page
2. grammar page
3. reading page
4. listening page
5. writing page
6. quiz page
7. week index page
8. styles
9. audio
10. root course index page if needed

## Step 5
Cross-link all important new content across multiple sections.

Also verify that previous-week content is used inside the new content itself:
- vocabulary example sentences
- grammar example sentences
- main reading passage
- main listening passage
- writing warm-ups and final writing test
- 50-question quiz
- random vocabulary drill

## Step 6
Run qa_checklist.

## Step 7
After generating the week, also output:
- full updated learner_state.md
- full updated vocabulary_bank.csv
- new week-XX-summary.md
