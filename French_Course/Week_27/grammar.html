Week 27 Grammar

Main target: formal complex sentence control. Review target: formal register.

Main Grammar Target

write a short professional message or report with controlled connectors and formal tone.

PatternUseExample
formal purposewhy the report existsCe rapport presente un changement.
analysis verbexplain informationLe texte analyse le probleme.
connector controlorganized formal writingCependant, une solution est proposee.
passive reviewformal processLe document est envoye par courriel.
recommendationnext stepIl est recommande de confirmer l heure.
summary + actionprofessional closeJe vous remercie de votre attention.
