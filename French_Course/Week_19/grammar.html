Week 19 Grammar

Main target: qui, que, and dont basics. Review target: adjective and noun linking.

Main Grammar Target

add one useful detail to a person, place, or object using qui, que, or dont in controlled sentences.

PatternUseExample
quiwho / that as subjectLa note qui aide Alex est courte.
quethat / which as objectLa note que je lis est claire.
dontwhose / of which / that ... aboutLe rendez-vous dont je parle est demain.
noun + adjective + quilinked descriptionLe petit cafe qui est pres du parc est calme.
review pronoun contrastrelative word is not le/la/lesJe lis la note que Samira a ecrite.
one detail onlykeep clauses shortMarc est un ami qui m'aide.

Review Grammar Target

Older grammar is not separate homework here. It appears inside the new examples, the reading, the listening, the writing prompts, and the quiz.

adjective and noun linking

Pattern Ladder

  1. La note qui aide Alex est courte.
  2. La note que je lis est claire.
  3. Le rendez-vous dont je parle est demain.
  4. Le cafe qui est pres du parc est calme.
  5. La note que je lis est utile.
  6. Le bus que je prends est en retard.

Watch out: Use qui when the next verb needs a subject. Use que when the next part already has a subject. Treat dont as a recognition chunk for now.
