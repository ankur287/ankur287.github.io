Week 1 Writing

Write short, accurate sentences using the Week 1 vocabulary bank.

1. Sentence warm-up

  1. Complete: Bonjour, je m'________ Alex.
  2. Complete: Je ________ étudiant.
  3. Order: sur la table / est / le livre
  4. Order: en classe / nous / sommes / lundi
Reveal answers
  1. appelle
  2. suis
  3. Le livre est sur la table.
  4. Nous sommes en classe lundi.

2. Guided writing

  • Bonjour, je m'appelle Alex.
  • Je suis étudiant / étudiante.
  • Je suis en classe lundi.
  • Le livre est sur la table.
  • La question est facile.
  • Merci. Au revoir.

3. Controlled writing

Write 5 short sentences. Include a greeting, your name, one class sentence, one object sentence, and a closing.

4. Writing test

Task: Write a 5-7 sentence self-introduction for class. Use at least six Week 1 vocabulary items.

Support bank: bonjour, je m'appelle, je suis, étudiant, en classe, lundi, livre, stylo, table, facile, merci, au revoir

5. Checklist before reveal / submit

  • Did I start with a greeting?
  • Did I use je m'appelle?
  • Did I use je suis correctly?
  • Did I include one classroom object?
  • Did I finish with a courtesy word or goodbye?

6. Model answer hidden by default

Show model answer

Bonjour. Je m'appelle Alex. Je suis étudiant. Je suis en classe lundi. Le livre est sur la table. La question est facile. Merci. Au revoir.

Writing rubric

Task completionIncludes 5-7 sentences and all requested parts.
Vocabulary useUses Week 1 words in meaningful sentences.
Grammar controlUses subject pronouns and être forms correctly.
ClarityShort, clear, easy-to-check sentences.
