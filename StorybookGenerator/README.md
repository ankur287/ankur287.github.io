# Storybook Generator

Python package scaffold for storybook generation tools.

## Setup

```bash
source .venv/bin/activate
python -m pip install -U pip
python -m pip install -e .
```

## Add Packages

Add runtime dependencies to `dependencies` in `pyproject.toml`, then reinstall:

```bash
python -m pip install -e .
```

For one-off installation into the current virtual environment:

```bash
python -m pip install package-name
```
